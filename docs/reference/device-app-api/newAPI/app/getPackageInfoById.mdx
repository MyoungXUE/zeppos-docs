# getPackageInfoById

> Start from API_LEVEL `4.0` . Please refer to [API_LEVEL](../../../../guides/framework/device/compatibility.md).

Get some of the fields in the Mini Program configuration `app.json` by app ID.

## Type

```ts
function getPackageInfoById(option: Option): Result
```

## Parameters

### Option

| Property | Type                | Required | DefaultValue | Description     | API_LEVEL |
| -------- | ------------------- | -------- | ------------ | --------------- | --------- |
| appId    | <code>number</code> | Y        | -            | Mini Program ID | 4.0       |

### Result

| Type                | Description                                          |
| ------------------- | ---------------------------------------------------- |
| <code>object</code> | Please see the fields in `app.json` for more details |

## Example

```js
import { getPackageInfoById } from '@zos/app'

const packageInfo = getPackageInfoById({ appId: 1001 })
console.log(packageInfo.name)
```
